# ClockJS-Retro

**ClockJS-Retro** is a basic Clock made with **HTML, CSS & JavaScript** in a Retro / Old School Terminal Color Scheme.

This project makes the use of following :

- ES6 Syntax
- CSS KeyFrame Animations
